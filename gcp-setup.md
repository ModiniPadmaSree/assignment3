Create an account in Google Cloud Platform
1. Navigate to the GCP Website: Go to the official Google Cloud homepage.
2. Start the Free Trial: Click the "Get started for free" or "Try Free" button.
3. Sign Up: Sign in with an existing Google account (like Gmail) or create a new one if needed.
4. Complete Registration: Follow the on-screen prompts to provide your details, including:
5. Selecting your country.
6. Accepting the Google Cloud Terms of Service.
7. Providing a valid payment method (credit/debit card) for identity verification. Charges only apply after free trial ends.
8. Activate Your Account: Complete any final verification steps. A small authorization request might appear on your card statement, which will be refunded

Create a GCE instance
1. Navigate to Compute Engine & Sign in to the Google Cloud Console.
2. In the top-left corner, click the Navigation m.
3. Scroll down and select Compute Eng,ne, then VM instances.
4. Click the Create Instance button at the top of the VM instances page.
5. Configure name: Enter a unique name for your VM instance.
6. Region and Zone: Select a geographic Region (e.g., us-central1) and a specific Zone within that region (e.g., us-central1-a) our latency or availability.
7. Machine configuration: Select the Series (e.g., E2 for general purpose) and Machine type (e.g., e2-medium, which determines vCPUs and memory).
8. In the Boot disk section, click Change.
9. Operating system: Choose your preferred operating system(Ubuntu) with a version.
10. Boot disk type and Size (GB): Configure the disk type and size as needed.
11. Click Select to save the boot disk configuration.
12. Under the Firewall section, check the boxes for Allow HTTP traffic and/or Allow HTTPS traffic.
13. Review all the changes made and click the create button.
